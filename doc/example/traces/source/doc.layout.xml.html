    <refpurpose>Specifies the overall document layout.</refpurpose>

  <refsynopsisdiv>
    <programlisting>
      <xi:include xmlns:xi="http://www.w3.org/2001/XInclude" parse="text" href="syn/doc.layout.xml"/></programlisting>

  <refsection>
    <title>Description</title>
    <para>The parameter configures the overall document layout, i.e. it
    specifies if the document must contain a coverpage, some TOC/LOTs,
    a frontmatter
    layout (for metadata/preface/dedication rendering), an index section,
    etc.</para>
    <para>See also the <parameter>doc.lot.show</parameter> and
    <parameter>doc.toc.show</parameter> parameters to specify the
    content of the TOC/LOTs.</para>
  </refsection>
