    <refpurpose>Raw options to pass to the geometry package</refpurpose>
  </refnamediv>
  <refsynopsisdiv>
    <programlisting>
      <xi:include xmlns:xi="http://www.w3.org/2001/XInclude" parse="text" href="syn/geometry.options.xml"/></programlisting>

  <refsection>
    <title>Description</title>
    <para>The content of this parameter is directly passed as options to the
    latex geometry package, when the page layout setup is customized by the
    user. Page layout customization is done through setting one or more of the
    pagination parameters (see below). If
    <parameter>geometry.options</parameter> is set but none of the other
    parameters is set, nothing is passed to geometry because the package is
    loaded only when the layout is not the default one.</para>
  </refsection>
